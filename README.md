
Apply the plugin to the maven pom.

```xml
<plugin>
    <groupId>uk.co.codezen</groupId>
    <artifactId>react-jsxtransformer-maven-plugin</artifactId>
    <version>1.0</version>

    <executions>
        <execution>
            <goals>
                <goal>compile</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```
